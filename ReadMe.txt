项目介绍
1，该项目使用C++11。所以编译要支持C++11，win上我用的是vs2013，linux上我用的是gcc/g++ 5.2
2，该项目支持windows平台作为守护程序
	在win上运行时，守护程序作为后台进程存在，看不到控制台窗口，但是被守护的程序窗口是存在的。
如果要关闭守护程序任务管理器杀进程或者自己下写批处理脚本关闭。
3，该项目支持linux平台作为守护程序
	在linux上，需要注意配置文件格式。
4，该守护进程支持同时监控多个应用程序
	至于支持几个是通过配置文件里count参数来配置的。
